# IpLookUp

## About Tool:
IpLookUp is a simply python script, which can be used to Ip lookup and to get information of perticualr target Ip. This tool works on both rooted Android device and Non-rooted Android device.

### REQUIREMENTS :
* internet
* storage 400 MB
* python
